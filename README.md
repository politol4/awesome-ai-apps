## Getting Started

### Prerequisites

- Python 3.9+
- [pip](https://pip.pypa.io/en/stable/) or [uv](https://github.com/astral-sh/uv)
- API keys for the relevant services (see individual app READMEs)

### Running an App

Each application lives in its own directory and includes a dedicated `README.md` with setup instructions. Navigate to the app's folder and follow the steps there:

```bash
# Clone the repository
git clone https://github.com/awesome-ai-apps/awesome-ai-apps.git
cd awesome-ai-apps

# Navigate to a specific app
cd <app-directory>

# Install dependencies
pip install -r requirements.txt

# Follow the app-specific README for configuration and running
```

## Contributing

Contributions are welcome! Whether it's a new app, bug fix, or documentation improvement — we'd love to have it.

Please read our contributing guidelines before submitting:

1. **Fork** the repository
2. **Create** a new branch (`git checkout -b feat/my-new-app`)
3. **Add** your app following the [README template](.github/README_TEMPLATE.md)
4. **Commit** your changes (`git commit -m 'feat: add my new app'`)
5. **Push** to your branch (`git push origin feat/my-new-app`)
6. **Open** a Pull Request using the provided template

### Reporting Issues

Found a bug or have a feature request? Please [open an issue](https://github.com/awesome-ai-apps/awesome-ai-apps/issues/new/choose) using one of the provided templates.
